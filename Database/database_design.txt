
Users Table with all users. Columns:
- ID (Primary Key);
- Email;
- Name;
- Phone Number;
- Status;
- Entry Mon;
- Entry Tue;
- Entry Wed;
- Entry Thu;
- Entry Fri;
- Entry Sat;
- Entry Sun;
- Exit Mon;
- Exit Tue;
- Exit Wed;
- Exit Thu;
- Exit Fri;
- Exit Sat;
- Exit Sun;

Each user will have a unique table for him, User History, that will store every time he enters and leaves the room. Columns:
- ID (Primary Key);
- Entry Date;
- Entry Time;
- Exit Date;
- Exit Time;
